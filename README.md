## Building and Running Examples

### Build entire project:
```bash
./gradlew build
```

### Build specific example:
Each example directory contains its own README.md with specific build and run instructions. Browse to the example directory and follow the provided instructions.

### Requirements

* Java 17 or higher
* Gradle (included via wrapper)
