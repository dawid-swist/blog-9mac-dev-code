# Java 17 Features Every Senior Developer Should Know

A comprehensive six-part series exploring essential Java 17+ features with practical examples and deep dives. Perfect for senior developers upgrading from Java 8 or anyone modernizing their Java knowledge.

## Series Parts

1. **[Part 1: Introduction & var Keyword](./part-1-introduction-and-var.md)**
   - Local variable type inference with the `var` keyword
   - Type inference with collections
   - Anonymous classes and var
   - Intersection types and limitations
   - Best practices for var usage

2. **[Part 2: Records - Modern Data Carriers](./part-2-records.md)**
   - Records as immutable data carriers
   - Generic records with type parameters
   - Custom methods and constructors
   - Implementing interfaces with records
   - Nested records and performance advantages

3. **[Part 3: Sealed Classes](./part-3-sealed-classes.md)**
   - Understanding sealed classes
   - Controlled inheritance hierarchies
   - Permitted subtypes and constraints
   - Sealed vs abstract classes

4. **[Part 4: Pattern Matching and Switch Expressions](./part-4-pattern-matching-and-switch.md)**
   - Pattern matching with instanceof
   - Enhanced switch expressions with arrow syntax
   - Modern switch with type patterns
   - Guard expressions

5. **[Part 5: Text Blocks](./part-5-text-blocks.md)**
   - Multi-line string literals (text blocks)
   - Escape sequences and formatting
   - String template processing
   - Use cases for JSON, SQL, and HTML

6. **[Part 6: Syntax Cheat Sheet](./part-6-syntax-cheat-sheet.md)**
   - Quick reference for all covered features
   - Code examples for each feature
   - Migration guide from Java 8
